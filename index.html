GitHub

        GitHub是一个来自微软的源代码公开平台，我在GitHub创建了多个项目，希望代码自由能够尽快实现。我认为，在虚拟世界中，人人平等，没有贫富差距、种族歧视和性别歧视等现象。那么，科学技术也应该公众于世，而不是作为资本的工具。信息技术是造福人类的，而不是为人类添乱的。我是一个代码自由主义者，也是一名共产主义者，希望我们能够贡献出自己的微薄力量。

        我的GitHub项目遵循GPL3.0开源协议，一切借助我的开源项目的原创代码或原创创意的开发者也必须遵循GPL3.0开源协议，将源代码开源，希望各位能够共同维护和平自由的网络代码环境。
