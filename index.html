         在2022年我经历了……，精彩的2022年即将逝去，这一年经历了很多的事情，没有人能像白纸一样没有故事，成长的代价就是失去原来的样子，但是我们应当燃起斗志，学会突破自我，发扬个性，我相信人生而平等，而我生来不凡。那就希望在即将到来的2023年，……，做出真正的自己！点击阅读全文>>






    

GitHub

        GitHub是一个来自微软的源代码公开平台，我在GitHub创建了多个项目，希望代码自由能够尽快实现。我认为，科学技术是造福人类的，那么，科学技术也应该公众于世，而不是作为资本的工具。我是一个代码自由主义者，也是一名共产主义者，希望能够贡献出自己的微薄力量。

        科技是站在前人的肩膀上发展，注册专利相当于拒绝其他人去使用拓展它；如果这个世界上充满了专利，以后后人想要解决一个简单的问题，都要当心某项专利的话，这不是一个我乐意生活的世界。

        我的GitHub项目均遵循此开源协议与版权声明，希望各位开发者能够共同维护和平自由的网络代码环境。
